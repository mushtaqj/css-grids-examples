Spanning Items Across Rows and Columns

Grid items span only one column and row track by default, but can span multiple row and/or column tracks using the same properties to position them. Set a grid item to span more than one column track by setting grid-column-end to a column line number that is more than one column away from grid-column-start. 
      grid-column-start: 1;
      grid-column-end: 4;
Grid items can also span across multiple row tracks by setting grid-row-end to more than one row track away. 
      grid-row-start: 1;
      grid-row-end:   4;
Shorthand properties grid-row and grid-column can also be used to position and span grid items more than one row or column. 
      grid-row:    2 / 5;
      grid-column: 2 / 4;
The keyword span, followed by the # of columns or rows to span, can also be used. 
      grid-row:    2 / span 3;
      grid-column: span 2;
 .wrapper {
    display: grid;
    grid-template-columns: 50%
  }

  .one {
    grid-column-start: 1;
    grid-column-end: 3;
    background: yellow;
  }
