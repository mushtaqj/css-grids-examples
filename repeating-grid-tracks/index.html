Repeating grid tracks

Define repeating grid tracks using the repeat() notation. This is useful for grids with items with equal sizes or many items. The repeat() notation accepts 2 arguments: the first represents the number of times the defined tracks should repeat, and the second is the track definition. repeat() can also be used within track listings. In this example, the first and last column tracks have widths of 30px, and the 3 column tracks in between, created by repeat(), have widths of 1fr each. 

    .wrapper {
      display: grid;
      grid-template-rows: repeat(4, 100px);
      grid-template-columns: repeat(3, 1fr);
    }
